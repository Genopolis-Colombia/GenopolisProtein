# Evidencia GA7-220501096-AA2-EV01

Este proyecto corresponde a la evidencia de desempeño “Codificación de módulos del software según requerimientos del proyecto”.

## Descripción
Se desarrolló el módulo de gestión de proteínas utilizando Java, aplicando una arquitectura por capas.  
El módulo permite realizar las operaciones básicas de persistencia de datos (CRUD) sobre una base de datos MySQL.

## Funcionalidades
- Crear proteína
- Consultar proteína
- Actualizar proteína
- Eliminar proteína

## Tecnologías utilizadas
- Java
- MySQL
- Gradle
- Git para versionamiento

## Estructura
El proyecto está organizado por capas, separando dominio, casos de uso, adaptadores y persistencia, facilitando el mantenimiento y escalabilidad del sistema.

## Pruebas
Las funcionalidades del módulo fueron probadas mediante endpoints REST y una colección de Postman incluida como soporte de pruebas.
